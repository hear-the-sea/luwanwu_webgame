## 技术栈速览

| 组件 | 技术 | 版本 |
|------|------|------|
| Web 框架 | Django | 5.x |
| REST API | Django REST Framework | 3.x |
| 异步通信 | Django Channels | 4.x |
| 任务队列 | Celery | 5.x |
| 消息代理 | Redis | 7.x |
| 数据库 | MySQL / SQLite | 8.x / 3 |
| 前端 | Django Templates + Bootstrap | 5 |

## 文档维护

### 文档更新原则

1. **同步更新** - 代码变更时同步更新相关文档
2. **版本标注** - 重大变更在文档中标注版本号
3. **示例完整** - 提供可运行的代码示例
4. **中文优先** - 面向中文开发者，专业术语保留英文

### 文档格式规范

- 使用 Markdown 格式
- 代码块标注语言类型
- 表格用于结构化数据展示
- 重要信息使用引用块或加粗
