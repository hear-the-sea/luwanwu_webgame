# HP系统重构总结

## 修改时间
2025-12-11

## 修改原因
取消升级的HP成长机制，改为完全由防御属性决定HP，以实现以下目标：
1. **强制玩家权衡攻防**：全加攻击会降低HP，全加防御会降低输出
2. **职业差异化**：武将天生防御高→血量高，文官天生防御低→血量低
3. **简化系统**：移除hp_growth相关的复杂计算逻辑
4. **增加策略深度**：鼓励多样化配队和装备选择

## 核心公式变更

### 旧系统
```python
max_hp = base_hp + hp_growth × (level - 1) + hp_bonus
```

示例（100级紫色武门客）：
- base_hp = 2400
- hp_growth = 170
- max_hp = 2400 + 170 × 99 = **19,230**

### 新系统
```python
max_hp = base_hp + defense × 50 + hp_bonus
```

示例（100级紫色武门客，防御264）：
- base_hp = 2400
- defense = 264
- max_hp = 2400 + 264 × 50 = **15,600**（全加武力）
- max_hp = 2400 + 363 × 50 = **20,550**（全加防御）

## 战术影响

### 1. 配点选择更重要
- **全攻型**：高输出但脆皮，适合速战速决
- **平衡型**：攻防兼备，主流选择
- **全防型**：超级坦克，适合前排吸伤

### 2. 装备系统价值提升
- 防御装备不仅提供减伤，还显著增加HP
- 攻击装备牺牲生存换取输出
- 装备选择成为战术核心

### 3. 职业差异更明显
- **武将**（基础防御82）：天生血厚，适合前排
- **文官**（基础防御70）：天生脆皮，需要保护

### 4. BOSS难度调整
- 独孤求败（50级）：37,800 → 43,200 HP（+14.3%）
- 高防御BOSS血量更高，更难击败
