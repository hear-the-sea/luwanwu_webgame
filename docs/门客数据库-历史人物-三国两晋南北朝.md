## 历史人物篇 - 三国两晋南北朝

> 包含：三国（魏蜀吴）、西晋、东晋、十六国、南北朝时期的历史人物

### 🟠 橙色

| 姓名 | 简介 | 类型 | 稀有度 |
|------|------|------|--------|
| 曹操 | （155年—220年3月15日）东汉末政治家、军事家、文学家，曹魏政权奠基者。挟天子以令诸侯，官渡之战击败袁绍并基本统一北方，推行屯田与唯才是举。建安时期文风兴盛，他亦擅诗文，著有《短歌行》《观沧海》等；曹魏建立后追尊为魏武帝。 | 武 | orange |
| 诸葛亮 | （181年—234年10月8日）蜀汉丞相，政治家、军事家。隆中对奠定蜀汉战略，辅佐刘备入蜀并平定南中；在成都主持内政，整饬吏治、发展农桑与军需。对外奉行联吴抗魏，多次北伐曹魏，卒于五丈原，著《出师表》。 | 文 | orange |
| 司马懿 | （179年—251年9月7日）曹魏重臣、军事统帅。镇守西线时多次与蜀汉北伐军对峙；亦曾东征平定辽东公孙渊。高平陵之变后夺取朝政大权，奠定司马氏取魏建晋的基础。 | 文 | orange |
| 刘裕 | （363年4月16日—422年6月26日）东晋末年军事家、政治家，南朝宋开国皇帝（宋武帝）。以北府兵起家，平定桓玄等内乱势力；北伐南燕、后秦等，入关收复长安并一度控制关中。其后取代东晋建立刘宋，开启南朝新局。 | 武 | orange |
| 拓跋焘 | （408年—452年3月11日）北魏太武帝。先后灭北燕、北凉并击退柔然，完成北方统一；在位时整饬政务并重用崔浩等辅臣。其统治后期曾下令禁佛，史称“太武灭佛”，并因宫廷变故而终致政局震荡。 | 武 | orange |

### 🟣 紫色

| 姓名 | 简介 | 类型 | 稀有度 |
|------|------|------|--------|
| 荀彧 | （163年—212年）王佐之才，汉室孤忠。他弃袁绍而投曹操，献策定兖州，坐镇后方运筹帷幄。官渡之战前后，他主掌军政机务与后方保障，多次进谏以定大计；然而当曹操欲进魏公时，他以死明志守护汉室最后的尊严。一个空食盒，道尽忠臣末路的悲凉。他是曹营第一谋主，也是最孤独的理想主义者。 | 文 | purple |
| 王猛 | （325年—375年）前秦名相，扪虱谈天下。他辅佐苻坚使前秦强盛一时，临终劝谏勿伐东晋。他是五胡十六国时期最杰出的政治家和军事家，被誉为"功盖诸葛第一人"。他若多活十年，天下或将一统。 | 文 | purple |
| 谢安 | （320年—385年10月12日）东山再起，淝水定倾。他隐居东山二十余载，一朝出山便挽狂澜于既倒。淝水之战，八万北府兵大破苻坚百万大军，他于棋局间接捷报，神色不改，只道"小儿辈大破贼"。那份从容淡定，是魏晋风度的极致。他以名士之姿担社稷之重，为东晋续命数十年。 | 文 | purple |
| 关羽 | （？—220年1月）青龙偃月、赤兔追风等形象多见于《三国演义》与传说。他于万军之中取颜良首级，解白马之围，威震华夏；水淹七军擒于禁、斩庞德为史实所载。至于“千里走单骑护嫂寻兄”等细节，多为演义渲染。虽败走麦城，英魂不灭，后世尊为武圣，与文圣孔丘并祀千秋。 | 武 | purple |
| 张飞 | （？—221年）燕人张翼德，万军辟易。他豹头环眼，声若雷霆，“当阳桥断喝”等事多见于《三国演义》。他粗中有细，义释严颜传为美谈；取西川立下汗马功劳。虽因暴而亡于小人之手，却无损其万夫莫敌之威名。蜀汉五虎，他与关羽并称"万人敌"。 | 武 | purple |
| 赵云 | （？—229年）常山赵子龙，蜀汉名将。长坂坡救幼主事迹最著名，演义中渲染为“七进七出”等；“枪挑高览、力斩五将”等亦属演义情节。一生追随刘备，以忠勇谨慎著称。 | 武 | purple |
| 吕布 | （？—199年2月7日）人中吕布，马中赤兔。“辕门射戟”“虎牢关”等多为《三国演义》渲染，方天画戟亦属演义常见形象。其人骁勇善战，然反复无常，先杀丁原后刺董卓，终为曹操所擒。白门楼上英雄末路，枭雄一世却难逃"三姓家奴"之讥。 | 武 | purple |
| 刘备 | （161年—223年6月10日）织席贩履出身，终成一方霸主。“桃园结义”“三顾茅庐”等为《三国演义》与民间叙述中的典型写法。半生颠沛流离，寄人篱下而志不改；入蜀后汉中称王，后称帝建立蜀汉。夷陵兵败，白帝托孤，临终犹念"勿以恶小而为之，勿以善小而不为"。 | 文 | purple |
| 孙权 | （182年—252年5月21日）生子当如孙仲谋。他承父兄基业，坐断东南战未休。赤壁联刘抗曹，火烧连营定三分；石亭诱敌深入，大破曹休震中原。他知人善任，周瑜、陆逊皆其所拔；他隐忍持重，在魏蜀夹缝中求得东吴五十年基业。紫髯碧眼，雄姿英发，纵横捭阖间尽显帝王手腕。 | 武 | purple |
| 周瑜 | （175年—210年）雄姿英发，羽扇纶巾。他弱冠之年便统兵征战，赤壁一役一把火大破曹操大军，奠定三分天下之基。谈笑间樯橹灰飞烟灭，顾曲间尽显名士风流。世人误传"既生瑜何生亮"，殊不知他与诸葛亮惺惺相惜，皆是一时之杰。英年早逝，壮志未酬，一曲长河落日，道尽千古遗憾。 | 武 | purple |
| 王羲之 | （303年—361年）书圣之名，千古独步。他于兰亭曲水流觞，挥毫写就天下第一行书。那二十一个"之"字各具姿态，飘若浮云矫若惊龙。他不慕权贵，辞官归隐，以鹅换字传为美谈。笔走龙蛇间，将魏晋风骨凝于纸上；墨香流转处，开后世书法之宗。千载之下，《兰亭序》仍是无数书家心中的圣殿。 | 文 | purple |
| 陈庆之 | （484年—539年）千军万马避白袍。他以书生之躯统七千白袍军，北伐四十七战，攻城三十二座，所向披靡。洛阳城下，以寡敌众破敌百万，创造了战争史上的不朽神话。他不是武夫，却让天下武人汗颜；他只是儒将，却书写了最壮烈的传奇。名师大将莫自牢，千军万马避白袍，梁朝一代战神，实至名归。 | 武 | purple |
| 祖冲之 | （429年—500年）圆周率精确千年，历法天文惊世间。他将圆周率推算至小数点后七位，领先世界近千年；编制《大明历》精确推算岁差，是中国古代最杰出的数学家天文学家。他不仅是科学巨匠，更造出指南车、千里船等奇械。在那个群雄逐鹿的时代，他以学问开辟了另一片天地。 | 文 | purple |
| 慕容恪 | （？—367年）前燕战神，国之柱石。慕容皝第四子，虽为庶出，却才兼文武。连环马阵大破冉闵，擒杀武悼天王，威震中原。辅佐幼主慕容暐，公正廉明，虚怀若谷。谢安曾言：“即使桓温北伐，若是遇上慕容恪，恐怕也难有胜算。”他是十六国时期最完美的统帅之一。 | 武 | purple |
| 高长恭 | （541年—573年）兰陵王，面具战神。北齐名将，貌柔心壮。邙山之战率五百骑突围，威震敌胆。因功高被鸩杀，并以悲剧英雄著称。 | 武 | purple |
| 鲁肃 | （172年—217年）联刘抗曹的战略家，东吴远见者。他最早提出三分天下之策（榻上策），力主孙刘联盟对抗曹操。赤壁之战他是幕后推手，虽非将帅却决定了战争走向。他为人方正，深谋远虑，是东吴霸业的规划师。 | 文 | purple |
| 陆逊 | （183年—245年3月19日）书生拜大将，火攻定乾坤。他以白面书生之貌麻痹关羽，白衣渡江奇袭荆州；夷陵之战隐忍不发，待刘备连营七百里疲惫之时，一把火烧尽蜀汉复仇大军。他是东吴后期的擎天柱石，文韬武略皆臻化境。石亭破曹休，镇守荆襄数十年，功业不在周瑜之下。 | 武 | purple |
| 孙策 | （175年—200年5月5日）小霸王横扫江东，霸业初成身先死。他继承父志，以传国玉玺换得兵马，横扫六郡席卷江东。神亭岭单挑太史慈，豪气干云；临终托孤于周瑜张昭，远见卓识。若非遇刺英年早逝，或许三分天下将是另一番格局。他是乱世中最耀眼的流星，虽短暂却璀璨夺目。 | 武 | purple |
| 郭嘉 | （170年—207年）鬼才早逝，遗计定辽东。他料敌如神，十胜十败激励曹操；遗计安辽东，令二袁自相残杀。若非英年早逝，赤壁之败或可避免。曹操哭祭道"哀哉奉孝，痛哉奉孝"，足见其在曹营中无可替代的地位。他是乱世中最耀眼的谋士流星，虽短暂却光芒万丈。 | 文 | purple |
| 吕蒙 | （178年—220年）士别三日，刮目相看。他从吴下阿蒙成长为东吴大都督，白衣渡江奇袭荆州，生擒关羽威震天下。读书破万卷的他，用实际行动证明：武将亦可运筹帷幄。可惜功成之后暴病而亡，为这段传奇添上了几分神秘色彩。他是东吴由武转文的典范，也是关羽陨落的关键人物。 | 武 | purple |
| 贾诩 | （147年—223年8月11日）毒士之名，明哲保身。他献计李傕郭汜祸乱长安，为张绣谋划宛城之变；归曹后韬光养晦，却在关键时刻一言定立储之争。他算无遗策却从不争功，是乱世中活得最通透的谋士。世人道他毒，却不知他只是比所有人都更懂得生存之道。 | 文 | purple |
| 邓艾 | （197年—264年2月）偷渡阴平，奇兵灭蜀。他结巴口吃，却是三国后期最杰出的军事家。率偏师翻越摩天岭，从阴平小道直插蜀汉腹地，逼降刘禅一战灭国。功高震主，却死于钟会谗言，父子俱亡于蜀中。他的结局证明：功成身退才是乱世生存的智慧。 | 武 | purple |
| 慕容垂 | （326年—396年6月2日）战神归来，后燕开国。他本是前燕名将，亡国后辅佐苻坚。淝水之败后复国建立后燕，年过六旬仍能征战沙场，一生未尝一败。他是十六国时代最杰出的军事家之一。 | 武 | purple |
| 桓温 | （312年—373年7月14日）北伐名将，权臣枭雄。他三次北伐收复洛阳，威震天下。晚年权势滔天欲行废立，终因声名所累未能篡位。他是东晋最矛盾的人物——有功亦有过。 | 武 | purple |
| 谢玄 | （343年—388年）北府兵魂，淝水先锋。谢安之侄，组建北府兵。淝水之战指挥若定，大破前秦。文武兼备，战后名望甚隆。 | 武 | purple |
| 宇文邕 | （543年—578年）北周武帝，灭佛雄主。隐忍多年，一朝诛杀权臣宇文护，夺回朝政。厉行改革，整顿吏治，释放奴婢。发起灭佛运动，将寺院财富充作军资，极大地增强了国力。灭北齐统一北方，为后来隋唐大一统奠定了坚实基础。并以英明果决著称，可惜英年早逝。 | 武 | purple |
| 斛律光 | （515年—572年）落雕都督，北齐长城。出身将门，善骑射，曾射落云中大雕，人称“落雕都督”。镇守边疆数十年，威震北周，宇文泰、宇文邕皆忌惮三分。然而因“百升飞上天，明月照长安”的谣言，被昏君高纬冤杀，北齐自毁长城，不久便亡。 | 武 | purple |
| 韦孝宽 | （509年—580年）玉壁铁壁，谍战鼻祖。镇守玉壁，面对高欢数十万大军围攻，用心理战和防御战术坚守六十日，耗死一代枭雄高欢。善于用间谍，在此用反间计除掉劲敌斛律光。一生南征北战，智谋深远，是北周首屈一指的名将。 | 武 | purple |
| 高颎 | （541年—607年）隋朝良相，真宰相材。辅佐隋文帝开创“开皇之治”，推行输籍法，整顿户籍，富国强兵。他文武双全，灭陈之战中运筹帷幄。因反对废太子杨勇而被贬，后被炀帝杀害。史称“隋之兴亡，系于高颎一人”。 | 文 | purple |
| 苏绰 | （498年—546年）六条诏书，改革先驱。辅佐宇文泰改革政治，制定“六条诏书”，确立了北周的政治纲领。创制计账、户籍法，精简机构，裁汰冗员。他的改革为北周的强盛和后来的隋唐制度奠定了坚实基础。 | 文 | purple |
| 法正 | （176年—220年）蜀汉第一谋主，刘备之郭嘉。他助刘备取益州、定汉中，定军山之战由他运筹策划。诸葛亮曾叹若法正在，必能劝阻东征。可惜天不假年，英年早逝。他是刘备最信任的谋士，若能长寿蜀汉或有不同的命运。 | 文 | purple |
| 钟会 | （225年—264年3月3日）书法名家之子，伐蜀主将。他才华出众却野心勃勃，灭蜀后妄图割据自立。最终谋反失败身死蜀中，连累邓艾一同遇害。他是才智与野心俱高的悲剧人物。 | 武 | purple |
| 崔浩 | （381年—450年）北魏谋主，国史之狱。辅佐太武帝统一北方，多谋善断，算无遗策。因修国史直书鲜卑恶迹，触怒鲜卑贵族，因此族灭。才高震主，结局惨烈。 | 文 | purple |
| 杨素 | （544年—606年）隋朝权臣，战舰千艘。造大舰灭陈，功勋赫赫。辅佐隋炀帝夺位，权倾朝野。虽有才干，然并以奸诈著称。 | 文 | purple |

### 🔵 蓝色

| 姓名 | 简介 | 类型 | 稀有度 |
|------|------|------|--------|
| 羊祜 | （221年—278年12月27日）灭吴谋划者，仁者无敌。他镇守荆州与陆抗对峙，以德服人令敌人敬佩。临终献灭吴之策，虽未亲见统一却是最大功臣。襄阳百姓为他立碑，名曰"堕泪碑"。他是西晋的柱石。 | 文 | blue |
| 陆抗 | （226年—274年）陆逊之子，东吴最后的名将。他与晋将羊祜隔江对峙，二人惺惺相惜传为佳话。他苦撑东吴危局，多次击退晋军进攻。他死后不久东吴便亡，他是东吴最后的屏障。 | 武 | blue |
| 马超 | （176年—222年）西凉锦马超，神威天将军。他出身将门，羌氐畏服，潼关之战杀得曹操割须弃袍。那杆虎头湛金枪挥洒自如，与张飞大战三百合不分胜负。虽后归刘备位列五虎，却郁郁而终于蜀中。他是西凉的骄傲，也是乱世中漂泊无依的孤狼，一生壮烈而悲凉。 | 武 | blue |
| 黄忠 | （？—220年）老当益壮，定军山上斩夏侯。他年近七旬仍能开三石硬弓，百步穿杨箭无虚发。长沙城下与关羽战成平手，定军山上阵斩夏侯渊威震天下。世人皆道廉颇老矣，却不知黄汉升越老越勇。他证明了英雄不问年龄，壮志不减当年，是蜀汉军中最励志的传奇。 | 武 | blue |
