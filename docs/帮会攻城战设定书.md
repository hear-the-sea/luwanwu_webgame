## 一、功能概述

帮会攻城战是**多人协作PVE**玩法，帮会成员联合攻打NPC城池，获取稀有奖励。此功能以帮会为单位参与，鼓励玩家加入帮会并积极协作。

## 三、参与机制

### 3.1 报名阶段

**报名条件**：
- 帮会等级 ≥ 城池要求等级
- 帮会资源足够支付报名费
- 当前没有正在进行的攻城战
- 城池不在冷却中

**报名流程**：
```
1. 帮主/管理员选择目标城池
2. 支付报名费（帮会资源）
3. 设置开战时间（报名后1-24小时内）
4. 帮会公告通知所有成员
```

**报名费用**：
| 城池难度 | 银两 | 粮食 | 金条 |
|----------|------|------|------|
| ★☆☆☆☆ | 10000 | 5000 | 0 |
| ★★☆☆☆ | 30000 | 15000 | 1 |
| ★★★☆☆ | 80000 | 40000 | 3 |
| ★★★★☆ | 200000 | 100000 | 10 |
| ★★★★★ | 500000 | 250000 | 30 |

### 3.2 参战阶段

**参战条件**：
- 是该帮会的活跃成员
- 拥有至少1名 `idle` 状态的门客
- 拥有至少1种护院

**参战流程**：
```
1. 成员在开战前进入攻城战页面
2. 选择出战门客（最多5名）
3. 选择出战护院配置
4. 点击「参战」确认
5. 等待开战时间到达
```

**参战限制**：
- 每个成员只能参战一次（不可中途退出再加入）
- 参战门客在战斗期间状态变为 `deployed`
- 参战护院从 `PlayerTroop` 中扣除

### 3.3 战斗阶段

**战斗流程**：
```
开战时间到达
    ↓
第一波守军出现
    ↓
所有参战成员的部队 vs 守军
    ↓
守军全灭 → 下一波
    ↓
最后一波（含BOSS）
    ↓
BOSS击杀 → 攻城成功
或
时间耗尽/全军覆没 → 攻城失败
```

**伤害计算**：
- 使用现有战斗系统（`battle/services.py`）
- 每个成员的部队独立计算伤害
- 伤害累计到城池总HP上
- 记录每个成员的伤害贡献

**实时显示**：
- 城池剩余HP（进度条）
- 当前波次
- 各成员伤害排行
- 剩余时间

## 四、奖励系统

### 4.1 奖励分配原则

采用**方案D（各自独立奖励）**：
- 每个参战成员独立获得一份奖励
- 不分配、不稀释
- 按贡献系数调整奖励数量

### 4.2 个人奖励计算

```
个人奖励 = 基础奖励 × 贡献系数

贡献 = 个人造成伤害 / 城池总HP × 100%

贡献系数：
- 贡献 < 5%：0.5（防挂机）
- 贡献 5%-15%：0.8
- 贡献 15%-25%：1.0
- 贡献 25%-40%：1.2
- 贡献 > 40%：1.5（MVP）
```

**示例**（黑风寨，基础奖励20经验果）：

| 成员 | 伤害贡献 | 贡献系数 | 获得经验果 |
|------|---------|---------|-----------|
| 玩家A | 45% | 1.5 | 30 |
| 玩家B | 25% | 1.2 | 24 |
| 玩家C | 15% | 1.0 | 20 |
| 玩家D | 10% | 0.8 | 16 |
| 玩家E | 5% | 0.8 | 16 |

### 4.3 帮会奖励

攻城成功后，帮会获得：
- 帮会资源（银两、粮食、金条）
- 帮会声望（用于帮会排行）【补充】

### 4.4 额外奖励

**首杀奖励**：
- 帮会首次攻克某城池，所有参战成员额外获得50%奖励

**MVP奖励**：
- 伤害贡献最高的成员额外获得：
  - 稀有装备宝箱 ×1
  - 帮会贡献 +500

**全员存活奖励**【补充】：
- 若无任何门客阵亡，所有成员额外获得20%奖励

### 4.5 奖励物品

| 城池难度 | 经验果 | 银两 | 装备宝箱 | 特殊掉落 |
|----------|--------|------|---------|---------|
| ★☆☆☆☆ | 20 | 5000 | 绿色×1 | - |
| ★★☆☆☆ | 40 | 10000 | 绿色×2 | 蓝色装备(10%) |
| ★★★☆☆ | 80 | 25000 | 蓝色×1 | 紫色装备(5%) |
| ★★★★☆ | 150 | 50000 | 蓝色×2 | 紫色装备(15%) |
| ★★★★★ | 300 | 100000 | 紫色×1 | 橙色装备(5%) |

## 五、战斗机制

### 5.1 波次机制

每个城池有多个波次，难度递增：

```
波次1：普通守军
    ↓ 全灭后
波次2：精英守军
    ↓ 全灭后
波次3：BOSS + 护卫
```

**波次间隔**：
- 每波结束后有10秒准备时间
- 可用于调整阵容（但不能更换门客）

### 5.2 BOSS机制

**BOSS特性**：
- 高HP、高攻击、高防御
- 拥有特殊技能
- 技能有冷却时间

**BOSS技能示例**：
```yaml
- key: aoe_attack
  name: 横扫千军
  description: 对所有敌方单位造成伤害
  damage: 150%
  cooldown: 3回合

- key: enrage
  name: 狂暴
  description: HP低于30%时触发，攻击力+50%
  trigger: hp_below_30%

- key: summon
  name: 召唤援军
  description: 召唤一波小兵
  cooldown: 5回合
```

### 5.3 死亡与复活

**门客死亡**：
- 门客HP归零后退出战斗
- 战斗结束后HP恢复至1（重伤状态）
- 不影响其他成员继续战斗

**全军覆没**：
- 某成员所有门客和护院全灭
- 该成员退出战斗，但仍计算已造成的伤害
- 其他成员继续战斗

**攻城失败条件**：
- 所有参战成员全军覆没
- 或战斗时间耗尽

```python
# 城池模板
class SiegeTemplate(models.Model):
    key = models.SlugField("城池标识", unique=True)
    name = models.CharField("城池名称", max_length=64)
    description = models.TextField("描述", blank=True)
    difficulty = models.PositiveIntegerField("难度等级", default=1)  # 1-5星

    # 城池属性
    total_hp = models.PositiveIntegerField("总血量", default=100000)
    defense = models.PositiveIntegerField("防御力", default=100)

    # 守军配置
    defender_waves = models.JSONField("守军波次配置", default=list)

    # 奖励配置
    guild_rewards = models.JSONField("帮会奖励", default=dict)
    personal_base_rewards = models.JSONField("个人基础奖励", default=dict)
    drop_table = models.JSONField("掉落表", default=dict)

    # 要求
    required_guild_level = models.PositiveIntegerField("要求帮会等级", default=1)
    min_participants = models.PositiveIntegerField("最少参与人数", default=3)
    max_participants = models.PositiveIntegerField("最多参与人数", default=10)

    # 时间配置
    duration = models.PositiveIntegerField("战斗时长上限(秒)", default=1800)
    cooldown = models.PositiveIntegerField("冷却时间(秒)", default=86400)

    # 报名费
    entry_cost = models.JSONField("报名费用", default=dict)


# 攻城战实例
class SiegeRun(models.Model):
    class Status(models.TextChoices):
        RECRUITING = "recruiting", "报名中"
        PREPARING = "preparing", "准备中"
        BATTLING = "battling", "战斗中"
        VICTORY = "victory", "胜利"
        DEFEAT = "defeat", "失败"

    guild = models.ForeignKey("guilds.Guild", on_delete=models.CASCADE, related_name="siege_runs")
    siege = models.ForeignKey(SiegeTemplate, on_delete=models.CASCADE, related_name="runs")

    # 状态
    status = models.CharField(choices=Status.choices, default=Status.RECRUITING)
    current_wave = models.PositiveIntegerField("当前波次", default=1)
    current_hp = models.PositiveIntegerField("城池当前HP", default=0)

    # 时间
    scheduled_at = models.DateTimeField("预定开战时间")
    started_at = models.DateTimeField("实际开始时间", null=True, blank=True)
    completed_at = models.DateTimeField("完成时间", null=True, blank=True)

    # 统计
    total_damage = models.PositiveIntegerField("总伤害", default=0)

    created_at = models.DateTimeField(auto_now_add=True)


# 攻城战参与者
class SiegeParticipant(models.Model):
    siege_run = models.ForeignKey(SiegeRun, on_delete=models.CASCADE, related_name="participants")
    member = models.ForeignKey("guilds.GuildMember", on_delete=models.CASCADE)

    # 出战配置
    guests = models.ManyToManyField("guests.Guest", related_name="siege_participations")
    troop_loadout = models.JSONField("护院配置", default=dict)

    # 战斗统计
    damage_dealt = models.PositiveIntegerField("造成伤害", default=0)
    contribution_percent = models.FloatField("贡献百分比", default=0)

    # 奖励
    rewards_claimed = models.BooleanField("已领取奖励", default=False)
    rewards_data = models.JSONField("奖励数据", default=dict)

    joined_at = models.DateTimeField(auto_now_add=True)


# 攻城战战报
class SiegeReport(models.Model):
    siege_run = models.OneToOneField(SiegeRun, on_delete=models.CASCADE, related_name="report")

    # 战斗记录
    waves = models.JSONField("波次记录", default=list)
    damage_ranking = models.JSONField("伤害排行", default=list)

    # 结果
    is_victory = models.BooleanField("是否胜利", default=False)
    total_time = models.PositiveIntegerField("总用时(秒)", default=0)

    created_at = models.DateTimeField(auto_now_add=True)
```

```python
@shared_task
def check_siege_start():
    """检查并启动到时间的攻城战"""
    now = timezone.now()
    pending_runs = SiegeRun.objects.filter(
        status=SiegeRun.Status.PREPARING,
        scheduled_at__lte=now
    )
    for run in pending_runs:
        start_siege_battle.delay(run.id)


@shared_task
def start_siege_battle(run_id):
    """启动攻城战战斗"""
```
