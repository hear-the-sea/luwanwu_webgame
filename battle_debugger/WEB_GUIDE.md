# 战斗调试器 - 网页版使用指南

## 🚀 快速开始

### 1. 启动开发服务器

```bash
python manage.py runserver
```

### 2. 访问网页界面

打开浏览器访问: **http://127.0.0.1:8000/debugger/**

## 📖 功能介绍

### 首页 (`/debugger/`)

显示所有可用的预设配置和功能入口：
- **战斗模拟** - 运行单次或多次战斗
- **参数调优** - 对比不同参数值的影响
- **预设配置列表** - 查看和选择预设配置

### 战斗模拟 (`/debugger/simulate/`)

#### 功能特点
- 选择预设配置
- 自定义随机种子（用于复现）
- 设置重复次数（1-100次）
- 覆盖任意战斗参数

#### 可调参数
1. **屠戮倍率** (slaughter_multiplier) - 门客打小兵的伤害倍率
2. **小兵打门客攻击除数** (troop_attack_divisor_vs_guest)
3. **小兵打小兵攻击除数** (troop_attack_divisor_vs_troop)
4. **小兵防御除数** (troop_defense_divisor)
5. **五行相克倍率** (counter_multiplier)
6. **暴击率** (crit_chance)

#### 使用步骤
1. 选择预设配置（如"简单小兵测试"）
2. （可选）设置随机种子
3. 设置重复次数
4. （可选）覆盖参数值
5. 点击"开始模拟"

### 战斗结果 (`/debugger/result/<id>/`)

展示战斗模拟结果：

#### 统计信息
- 总战斗次数
- 攻方/守方胜利次数和胜率
- 平均回合数
- 胜率分布图（进度条）

#### 详细战斗日志（单次模拟时）
- 每回合的攻击事件
- 伤害和击杀数
- 技能触发
- 暴击标记

#### 结果列表（多次模拟时）
- 表格形式展示所有战斗
- 每场战斗的种子、胜者、回合数、兵损

### 参数调优 (`/debugger/tune/`)

对比单个参数的不同值，找到最优配置。

#### 使用步骤
1. 选择预设配置
2. 选择要调优的参数（如"屠戮倍率"）
3. 输入参数值列表（如：20,30,40,50）
4. 设置每个值的重复次数（建议10-20次）
5. 点击"开始调优"

### 调优结果 (`/debugger/tune_result/`)

展示参数调优的对比结果：

#### 对比表格
- 每个参数值的攻守胜率
- 平均回合数
- 战斗详情

#### 可视化图表
- **胜率变化曲线图** - 查看参数对胜率的影响
- **平均回合数柱状图** - 查看战斗时长变化

### 预设详情 (`/debugger/presets/<name>/`)

查看预设配置的详细信息：
- 攻守双方的门客配置
- 小兵数量和类型
- 科技等级
- 调优参数设置

## 💡 使用示例

### 示例1: 快速测试基础战斗

1. 访问首页，点击"战斗模拟"
2. 选择"简单小兵测试"
3. 设置重复次数为1
4. 点击"开始模拟"
5. 查看详细战斗日志

### 示例2: 测试屠戮倍率影响

1. 访问首页，点击"参数调优"
2. 选择预设"balanced"
3. 选择参数"屠戮倍率"
4. 输入值列表: `20,25,30,35,40`
5. 设置重复次数为20
6. 点击"开始调优"
7. 查看胜率曲线图

### 示例3: 验证特定配置

1. 访问首页，选择预设"boss_fight"
2. 点击"查看详情"查看配置
3. 点击"使用此配置"
4. 覆盖参数（如屠戮倍率=50）
5. 设置种子=12345（可复现）
6. 运行模拟
7. 查看结果

### 示例4: 批量测试统计

1. 访问"战斗模拟"
2. 选择"troop_heavy"
3. 设置重复次数=50
4. 不设置种子（随机）
5. 运行模拟
6. 查看统计信息和胜率分布

## 🚨 注意事项

1. **缓存时效**: 结果缓存1小时后过期，记得保存重要数据
2. **并发限制**: 建议单次重复不超过100次
3. **参数范围**: 注意参数的合理范围，避免极端值
4. **浏览器兼容**: 推荐使用Chrome、Firefox、Edge等现代浏览器
5. **调优时间**: 参数调优时间 = 重复次数 × 参数值数量 × 单次战斗时间

## 🎯 最佳实践

### 快速测试
- 使用重复次数=1，查看详细日志
- 设置固定种子，便于复现问题

### 统计分析
- 使用重复次数=10-50，获取可靠统计
- 不设置种子，测试随机性影响

### 参数调优
- 先用粗粒度值（间隔大），快速定位范围
- 再用细粒度值（间隔小），精确找到最优值
- 重复次数建议15-30次

### 问题排查
- 使用特定种子复现问题
- 查看详细战斗日志定位原因
- 逐步调整单个参数确定影响

## 📝 常见问题

**Q: 结果显示"结果不存在或已过期"？**
A: 结果缓存1小时后会过期，需要重新运行模拟。

**Q: 参数调优很慢？**
A: 减少重复次数或参数值数量，或使用更简单的预设。

**Q: 图表不显示？**
A: 检查网络连接（需要加载Chart.js），或刷新页面。

**Q: 如何分享结果？**
A: 复制结果页面的URL，缓存有效期内他人可访问。

**Q: 可以导出数据吗？**
A: 目前暂不支持，可以通过截图或复制数据。（后续版本可添加）
