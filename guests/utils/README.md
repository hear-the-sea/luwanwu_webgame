# Guests Utils 模块

游客系统的工具函数集合。

## 目录结构

```
guests/utils/
├── __init__.py           # 包初始化
└── name_generator.py     # 姓名生成工具
```

## 模块说明

### name_generator.py

负责为黑色和灰色稀有度的门客生成随机的中文古风姓名。

**功能特性：**
- 支持单姓/复姓、单字名/双字名、叠字名等多种组合
- 加权随机选择，常见姓氏出现概率更高
- 共320个姓氏（288个单姓 + 32个复姓）
- 共180+个名字字符，涵盖德行、自然、志向等多个类别

**概率分布：**
- 常见姓氏（李、王、张等60个）：≈63%
- 补充单姓（100个）：≈21%
- 历史姓氏（60个）：≈8.5%
- 地域/古代姓氏（68个）：≈5%
- 复姓（32个）：≈2.5%

**使用示例：**
```python
from guests.utils.name_generator import generate_random_name
import random

rng = random.Random()
name = generate_random_name(rng)
# 可能输出：王云飞、李明、司马懿、诸葛亮 等
```

## 数据结构

### BLACK_SURNAMES
包含320个姓氏的列表，按以下顺序排列：
1. 常见单姓（60个）
2. 补充单姓（100个）
3. 历史单姓（60个）
4. 地域姓氏（40个）
5. 古代姓氏（28个）
6. 复姓（32个）

### BLACK_GIVEN
包含180+个名字字符的列表，分类包括：
- 德行品格（仁、义、礼、智、信等）
- 自然意象（云、山、风、雨、松、竹等）
- 志向气概（远、志、勇、武、飞、翔等）
- 文采学识（文、学、诗、词、琴、棋等）
- 时令节气、玉石珍宝、位置方位等

### SURNAME_WEIGHTS
姓氏权重列表，用于实现加权随机选择：
- 常见单姓：权重 15
- 补充单姓：权重 3
- 历史单姓：权重 2
- 其他姓氏：权重 1

## 名字生成逻辑

1. **姓氏选择**：使用加权随机从320个姓氏中选择
2. **名字结构**：
   - 复姓：70%单字名 + 30%双字名
   - 单姓：40%单字名 + 60%双字名
3. **叠字名概率**：10%（如：云云、山山）
4. **避免重复**：双字名避免两字相同（叠字名除外）

## 维护说明

如需调整姓名生成策略，可修改以下参数：

1. **调整姓氏概率**：修改 `SURNAME_WEIGHTS` 中的权重值
2. **添加新姓氏/名字**：在 `BLACK_SURNAMES` 或 `BLACK_GIVEN` 中追加
3. **调整名字结构**：修改 `generate_random_name` 函数中的概率值
4. **调整叠字概率**：修改函数中的 `0.1` 概率值
